## How to Download from Project Gutenberg

1. Go to www.gutenberg.org
2. Search for book title
3. Look for "Plain Text UTF-8" download link
4. Save as `.txt` file with exact name from our list
5. Move to `C:\Users\chand\old_bibles\assets\texts\`

## How to Download from CCEL

1. Go to www.ccel.org
2. Browse or search for text
3. Look for "Plain Text" or "Download" option
4. Usually downloadable directly
5. Rename if necessary, move to assets folder

## Tips

- Many texts are freely available and identical across sources
- CCEL is best for religious/Christian texts (properly formatted)
- Gutenberg is best for literary/fiction works
- Save time by downloading multiple in parallel browser tabs
- Create a "download log" to track which files you've processed
- Some texts may need light cleanup (removing headers/footers)
